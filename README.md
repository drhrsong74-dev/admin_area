# 미추홀구 행정동별 상권분석 웹페이지

## GitHub Pages 배포
1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더 안의 `index.html`, `style.css`, `app.js`, `data.json`, `assets` 폴더를 저장소 최상위에 업로드합니다.
3. 저장소 `Settings → Pages`에서 `Deploy from a branch`, `main`, `/(root)`를 선택하고 저장합니다.
4. 생성된 GitHub Pages 주소로 접속합니다.

## 중요
- 원본 Excel과 HWPX 파일은 업로드하지 않아도 됩니다.
- 공개 저장소에는 웹 표시용 집계 데이터(`data.json`)와 차트 이미지만 포함됩니다.
- 파일을 PC에서 더블클릭하면 브라우저 보안 정책 때문에 `data.json`이 로드되지 않을 수 있습니다. GitHub Pages에서 확인하거나 VS Code Live Server를 사용하세요.
